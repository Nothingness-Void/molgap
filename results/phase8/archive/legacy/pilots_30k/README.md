# Legacy 30K Pilots

This directory contains the completed old30k/replacement30k/tail30k controlled
experiments: graph caches, encoder checkpoints and embeddings, end-to-end MoE,
fusion metrics, and common-eval outputs.

Decision: replacement coverage produced a weak positive hard-slice signal, but
MoE was tie-level and the tail probe did not justify a production route. The
later full 300K/500K experiments supersede these pilots. Do not use these files
as current model inputs.
