# MolGap プロジェクト進捗報告

市販有機電子材料分子の HOMO/LUMO/Gap 機械学習予測

---

## 1. プロジェクト概要

PubChemQC データベース（B3LYP/6-31G*//PM6）から分子構造のみで HOMO energy・LUMO energy・HOMO-LUMO gap を予測する機械学習モデルを構築した。

## 2. 訓練データ

| 項目 | 内容 |
|------|------|
| 分子数 | 30,000 |
| 元素範囲 | C, H, O, N, S, F, Cl（7種） |
| 分子量範囲 | 200–500 g/mol |
| データソース | PubChemQC B3LYP/6-31G*//PM6 |
| 単位 | eV |

## 3. モデル開発の推移

## 5. 汎化性能の検証

### 5.1 PubChemQC 未知分子での検証（100分子）

訓練データに含まれない100分子をランダム抽出し、予測値とデータベース計算値を比較。

| 予測対象 | MAE (eV) | R² | 誤差<0.2eV の割合 |
|----------|----------|------|-----------------|
| HOMO | 0.157 | 0.786 | 73% |
| LUMO | 0.192 | 0.906 | 64% |
| Gap | 0.221 | 0.863 | 60% |
| **平均** | **0.190** | **0.852** | — |

### 5.2 Gaussian 量子化学計算との比較（10分子）

市販OLED材料10分子に対し、Gaussian 16 B3LYP/6-31G(d) opt+freq 全最適化計算を実施し、ML予測値と比較。

| 分子 | 用途 | HOMO Δ (eV) | LUMO Δ (eV) | Gap Δ (eV) |
|------|------|-------------|-------------|------------|
| mCP | ホスト材料 | +0.06 | +0.00 | -0.06 |
| TCTA | 正孔輸送 | +0.02 | +0.14 | +0.12 |
| CBP | ホスト材料 | -0.06 | +0.19 | +0.24 |
| Coumarin-6 | 発光材料 | +0.12 | -0.17 | -0.29 |
| CzSi | ホスト材料 | +0.17 | -0.08 | -0.25 |
| BCP | 電子輸送 | +0.23 | -0.15 | -0.38 |
| TPBi | 電子輸送 | +0.29 | +0.32 | +0.03 |
| DPEPO | ホスト材料 | +0.35 | -0.00 | -0.35 |
| NPB | 正孔輸送 | -0.29 | +0.32 | +0.60 |
| Spiro-OMeTAD | 正孔輸送 | -0.58 | +0.60 | +1.19 |

| 予測対象 | ML vs Gaussian MAE (eV) |
|----------|------------------------|
| HOMO | 0.216 |
| LUMO | 0.196 |
| Gap | 0.352 |

※ Spiro-OMeTAD（MW=1225）は訓練データの分子量範囲（200-500）を大きく超えており、誤差が最大。除外した場合、Gap MAE ≈ 0.26 eV。

※ ML モデルは PubChemQC（PM6 最適化構造）で訓練、Gaussian 検証は B3LYP/6-31G(d) 全最適化構造を使用。計算手法の差異も誤差の一因。

## 6. 今後の方針

1. **PM6 最適化構造の利用**：PubChemQC に含まれる PM6 座標で SchNet を再訓練（現在は RDKit ETKDG 構造を使用）
2. **DimeNet++ / PaiNN**：角度情報を考慮する 3D GNN の導入
3. **データ拡大**：30k → 50k-100k への拡張
4. **市販分子データベース**：予測結果を用いた検索可能なデータベースの構築
