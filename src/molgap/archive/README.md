# Archived Experiment Modules

This package contains reusable implementation needed only to reproduce closed
experiments. Active training and inference code must not import from here.

- `phase8_r01_router/`: archive-r01 learned-Router and Late-Blend helpers.
- `phase8_r03_three_expert/`: archive-r03 three-expert MoE, Router, and loss helpers.
- `phase8_r04_static_dual2d/`: archive-r04 Local-GINE/GPS static-blend helpers.
- `phase8_r06_structural_gps_adapter/`: archive-r06 ring/conjugation G0 feature helpers.
