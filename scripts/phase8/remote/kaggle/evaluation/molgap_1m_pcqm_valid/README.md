# Fixed 1M PCQM Validity Evaluator

`pcqm_valid_1m.py` is the retained Kaggle adapter for evaluating the imported
1M model bundle on the fixed PCQM validity sample. It is an evaluator, not a
training entry point and not evidence that the 1M model is production-default.

Completed outputs are under
`results/kaggle/evaluation/runs/molgap_1m_pcqm4mv2_valid_v2/`.
